# 🧮 Scientific Calculator Web App

A modern **Scientific Calculator** built using HTML, CSS, and JavaScript with Light/Dark mode, advanced math functions, and a clean responsive UI.

---

## 🚀 Features

### 🔢 Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Parentheses support

---

### 📊 Scientific Functions
- sin, cos, tan  
- log (base 10), ln (natural log)  
- sqrt (square root)  
- Power (^ exponent)  
- Constants: π (pi), e  

---

### 🎯 Advanced Features
- Degree / Radian mode switch  
- Light / Dark theme 🌗  
- Dynamic background images  
- Calculation history 📜  
- Keyboard support ⌨️  
- Error handling for invalid inputs  

---

## 🎨 UI Features
- Glassmorphism modern design ✨  
- Fully responsive layout (mobile + desktop) 📱💻  
- Smooth animations and hover effects  
- Clean and professional UI  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
