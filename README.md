# 🧮 Scientific Calculator Web App

A modern **Scientific Calculator** built using HTML, CSS, and JavaScript with light/dark mode and advanced math functions.

---

## 🚀 Features

### 🔢 Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Parentheses support

### 📊 Scientific Functions
- sin, cos, tan
- log, ln
- sqrt (square root)
- power (^)
- constants: π, e

### 🎯 Advanced Features
- Degree / Radian mode
- Light / Dark theme
- Dynamic background images
- Calculation history
- Keyboard support

---

## 🎨 UI Features
- Glassmorphism design
- Responsive layout
- Smooth theme transitions
- Modern button animations
